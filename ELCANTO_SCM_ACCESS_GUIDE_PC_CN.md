
**方式B: 通过邮件附件**
```
如果文件通过邮件发送:

2. 查看附件部分
```

**方式C: 通过USB接收**
```
如果通过USB传输:
1. 将USB插入电脑
2. 打开USB驱动器
3. 复制 ProtonVPN_win.exe 到电脑
4. 建议复制到"下载"文件夹
```

**方式D: 通过Dropbox链接**

```
1. 点击公司发送的Dropbox链接
2. 点击"下载"按钮
3. 文件会下载到电脑
```

### 第2步: 安装VPN (Windows)

#### 安装前准备：
```
⚠️ 重要提示：
- 确保您有管理员权限
- 关闭杀毒软件（临时）
- 准备10分钟时间完成安装
```

#### 详细安装步骤：

**1. 找到下载的文件**
```
位置: C:\Users\您的用户名\Downloads\
文件名: ProtonVPN_win_v3.x.x.exe
文件大小: 约100-150MB
```

**2. 运行安装程序**
```
双击 ProtonVPN_win.exe 文件
```

**3. Windows安全提示**
```
如果出现"Windows已保护你的电脑"警告:
1. 点击"更多信息"
2. 点击"仍要运行"
3. 这是正常的安全提示，软件安全可靠
```

**4. 用户账户控制(UAC)提示**
```
如果弹出"是否允许此应用对设备进行更改":
点击"是"
```

**5. 安装向导**
```
1. 欢迎界面: 点击"Next"（下一步）
2. 许可协议: 选择"I agree"（我同意），点击"Next"
3. 安装位置: 使用默认位置，点击"Next"
4. 开始安装: 点击"Install"（安装）
5. 等待安装完成（约2-3分钟）
6. 完成: 点击"Finish"（完成）
```

### 第3步: 创建免费账户

**首次启动Proton VPN:**

```
1. 桌面双击 Proton VPN 图标
2. 欢迎界面会打开
3. 选择"Create Free Account"（创建免费账户）
```

**填写注册信息:**
```

       - 或163邮箱 (例: yourname@163.com)
       - 或公司邮箱

Password: 创建密码
          - 至少8个字符
          - 包含字母和数字
          - 记住此密码！

Confirm Password: 再次输入相同密码

点击"Create Account"（创建账户）
```

**验证邮箱:**
```
1. 打开您的邮箱
2. 查找Proton VPN验证邮件
3. 点击邮件中的验证链接
4. 返回Proton VPN应用
5. 使用邮箱和密码登录
```

## ❗ 常见问题解决

### 问题1: VPN无法安装

**错误: "Windows Defender已阻止"**
```
解决方案:
1. 右键点击 ProtonVPN.exe
2. 选择"属性"
3. 勾选"解除锁定"
4. 点击"确定"
5. 再次双击安装
```

**错误: "需要管理员权限"**
```
解决方案:
1. 右键点击 ProtonVPN.exe
2. 选择"以管理员身份运行"
3. 点击UAC提示的"是"
```

**错误: "安装程序已损坏"**
```
解决方案:
1. 重新下载文件
2. 确认文件完整下载完成
3. 检查文件大小是否正确（100-150MB）
4. 联系公司重新发送文件
```

### 问题2: VPN无法连接

**错误: "Connection failed"（连接失败）**
```
解决方案:
1. 尝试其他服务器
   - 日本 → 新加坡 → 韩国 → 荷兰
2. 检查网络连接是否正常
3. 关闭防火墙（临时测试）
4. 重启VPN应用
5. 重启电脑
```

**错误: 一直显示"Connecting..."（连接中）**
```
解决方案:
1. 等待1-2分钟
2. 点击"Disconnect"（断开）
3. 等待10秒
4. 重新选择服务器连接
```

**错误: "No internet connection"（无网络连接）**
```
解决方案:
1. 断开VPN
2. 测试正常网络是否可用
3. 重新连接VPN
4. 尝试其他服务器
```

### 问题3: 网站无法访问

**错误: "无法访问此网站"**
```
检查项:
□ VPN是否已连接（绿色图标）
□ 网址是否正确: https://elcanto-scm.web.app
□ 浏览器是否为最新版本

解决方案:
1. 确认VPN已连接
2. 清除浏览器缓存:
   - Chrome: Ctrl+Shift+Delete
   - 选择"所有时间"
   - 勾选"缓存"
   - 点击"清除数据"
3. 关闭浏览器，重新打开
4. 再次访问网站
```

**错误: 网站加载很慢**
```
解决方案:
1. 切换到日本或新加坡服务器
2. 关闭其他占用网络的程序
3. 等待2-3分钟
4. 刷新页面（F5键）
```

**错误: "Failed to get document because the client is offline"**
```
这说明Firebase被阻止了

解决方案:
1. 确认VPN已连接
2. 断开VPN，重新连接
3. 尝试其他服务器
4. 重启浏览器
5. 如果仍然失败，联系公司IT支持
```

### 问题4: 无法创建VPN账户

**错误: "Email already exists"（邮箱已存在）**
```
解决方案:
1. 使用该邮箱直接登录
2. 或点击"Forgot password"（忘记密码）重置
3. 或使用其他邮箱注册
```

**错误: "Invalid email"（无效邮箱）**
```
解决方案:
确保邮箱格式正确:
✅ yourname@qq.com
✅ yourname@163.com
✅ yourname@company.com
❌ yourname@qq (缺少.com)
❌ yourname@ (不完整)
```

## 💡 使用技巧

### 日常使用流程:

**每次使用ELCANTO系统:**
```
1. 打开Proton VPN
2. 连接到日本或新加坡服务器
3. 等待连接成功（绿色✅）
4. 打开浏览器
5. 访问 https://elcanto-scm.web.app
6. 登录使用
7. 使用完毕后，可以断开VPN
```

### 保持VPN稳定连接:

```
技巧:
1. 选择延迟最低的服务器
2. 不要频繁切换服务器
3. 如果断开，立即重新连接
4. 长时间不用建议断开节省资源
```

### 提高连接速度:

```
方法:
1. 使用有线网络（而非WiFi）
2. 关闭其他下载/上传任务
3. 选择日本或新加坡服务器
4. 避免高峰时段（晚上8-10点）
5. 重启路由器
```
