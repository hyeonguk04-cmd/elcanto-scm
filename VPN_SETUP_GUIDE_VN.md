# Hướng dẫn cài đặt VPN - Truy cập ELCANTO SCM

## Phương án 1: Cài đặt VPN qua file APK (Đề xuất)

### Các bước cài đặt:

1. **Tải file APK của VPN**
   - Tải file ProtonVPN.apk từ email công ty
   - Hoặc copy từ USB vào điện thoại

2. **Cho phép cài đặt từ nguồn không xác định**
   - Cài đặt → Bảo mật → Nguồn không xác định
   - Bật "Cho phép cài đặt ứng dụng không xác định"

3. **Cài đặt APK**
   - Tìm file ProtonVPN.apk đã tải
   - Nhấn vào để cài đặt
   - Làm theo hướng dẫn để hoàn tất

4. **Sử dụng VPN**
   - Mở Proton VPN
   - Tạo tài khoản miễn phí (nếu lần đầu sử dụng)
   - Chọn server: Nhật Bản, Singapore, hoặc Hàn Quốc
   - Nhấn "Kết nối"

5. **Truy cập website**
   - Sau khi VPN kết nối thành công
   - Mở trình duyệt
   - Truy cập: https://elcanto-scm.web.app
   - Đăng nhập bằng tên đăng nhập và mật khẩu được cung cấp

2. **Kết nối đến máy tính văn phòng Hàn Quốc**

3. **Sử dụng hệ thống trên máy tính từ xa**
