# VPN 다운로드 링크 및 파일 전송 가이드

## 📦 VPN 다운로드 링크

### Proton VPN (무료, 추천)

#### **Android (안드로이드)**
- **공식 사이트**: https://protonvpn.com/download-android
- **APK Mirror**: https://www.apkmirror.com/apk/proton-ag/proton-vpn/
- **APK Pure**: https://apkpure.com/proton-vpn/ch.protonvpn.android

**다운로드 방법**:
1. 위 링크 중 하나 접속 (중국 밖에서)
2. "Download APK" 클릭
3. 최신 버전 다운로드 (예: ProtonVPN_4.x.x.apk)
4. 파일 크기: 약 50-80MB

#### **Windows (윈도우)**
- **공식 사이트**: https://protonvpn.com/download-windows
- 파일 이름: ProtonVPN_win_v3.x.x.exe
- 파일 크기: 약 100-150MB

#### **Mac (맥)**
- **공식 사이트**: https://protonvpn.com/download-mac
- 파일 이름: ProtonVPN.dmg
- 파일 크기: 약 80-120MB

#### **iOS (아이폰/아이패드)**
- **App Store**: https://apps.apple.com/app/protonvpn/id1437005085
- 중국 App Store에서는 다운로드 불가
- 한국/미국 Apple ID 필요

## 📧 파일 전송 방법

### 방법 1: 이메일로 전송

**Gmail / Outlook / 네이버 메일**:
```
제목: ELCANTO SCM VPN 설치 파일
내용: 
안녕하세요,

ELCANTO SCM 시스템 접속을 위한 VPN 설치 파일입니다.

첨부 파일:
- ProtonVPN.apk (Android)
- 설치 가이드.pdf

자세한 설치 방법은 첨부된 가이드를 참고해주세요.

문의: yang_hyeonguk@elcanto.co.kr

감사합니다.
```

**주의사항**:
- Gmail 첨부 파일 제한: 25MB
- APK 파일이 25MB 넘으면 Google Drive 링크 사용
- 파일명에 한글 사용 금지 (예: ProtonVPN.apk ⭕, VPN설치.apk ❌)

### 방법 2: WeChat (微信)으로 전송 - 중국용

**단계**:
1. WeChat 앱 실행
2. 생산업체 담당자와 대화방 열기
3. "+" 버튼 → "파일" 선택
4. ProtonVPN.apk 선택
5. 전송

**팁**:
- WeChat 파일 전송 제한: 100MB
- 파일 유효기간: 7일 (다운로드 기한)
- 전송 후 "파일 전송 완료, 7일 내 다운로드 해주세요" 메시지

### 방법 3: QQ로 전송 - 중국용

**단계**:
1. QQ 앱 실행
2. 생산업체 담당자와 대화방 열기
3. "파일" 아이콘 클릭
4. ProtonVPN.apk 선택
5. 전송

### 방법 4: Zalo로 전송 - 베트남용

**단계**:
1. Zalo 앱 실행
2. 생산업체 담당자와 대화방 열기
3. "+" 버튼 → "File" 선택
4. ProtonVPN.apk 선택
5. 전송

**팁**:
- Zalo 파일 전송 제한: 50MB
- 파일 유효기간: 30일

### 방법 5: Google Drive / Dropbox 공유

**Google Drive**:
1. drive.google.com 접속
2. ProtonVPN.apk 업로드
3. 파일 우클릭 → "공유" → "링크 가져오기"
4. "링크가 있는 모든 사용자" 선택
5. 링크 복사하여 이메일/메신저로 전송

**Dropbox**:
1. dropbox.com 접속
2. 파일 업로드
3. "공유" → "링크 복사"
4. 링크 전송

**장점**:
- 파일 크기 제한 없음
- 만료 기한 없음 (또는 설정 가능)
- 여러 명에게 동시 전송 가능

### 방법 6: USB 드라이브

**중국 출장 시**:
1. USB에 다음 파일 복사:
   - ProtonVPN.apk
   - ELCANTO_SCM_ACCESS_GUIDE_CN.pdf
2. 중국 현지에서 직접 전달
3. 생산업체 PC/휴대폰에 복사

**장점**:
- 인터넷 불필요
- 대용량 파일도 가능
- 확실한 전달
