🌟 方法1: 安装VPN (Windows PC) ⭐⭐⭐⭐⭐

第1步: 获取VPN安装文件

从公司接收文件的方式：

方式A: 通过Google Drive链接下载

1. 公司会发送Google Drive链接给您
   例如: https://drive.google.com/file/d/xxxxx
2. 点击链接
3. 点击右上角"下载"按钮
4. 文件会下载到"下载"文件夹
5. 文件名: ProtonVPN_win_v3.x.x.exe
6. 文件大小: 约 100-150MB

方式B: 通过邮件附件

如果文件通过邮件发送:
1. 打开公司发送的邮件
2. 查看附件部分
3. 点击 ProtonVPN_win.exe 下载
4. 保存到电脑

方式C: 通过USB接收

如果通过USB传输:
1. 将USB插入电脑
2. 打开USB驱动器
3. 复制 ProtonVPN_win.exe 到电脑
4. 建议复制到"下载"文件夹

方式D: 通过Dropbox链接

1. 点击公司发送的Dropbox链接
2. 点击"下载"按钮
3. 文件会下载到电脑

第2步: 安装VPN (Windows)

⚠️ 安装前准备：

  • 确保您有管理员权限
  • 关闭杀毒软件（临时）
  • 准备10分钟时间完成安装

详细安装步骤：

1. 找到下载的文件

位置: C:\Users\您的用户名\Downloads\
文件名: ProtonVPN_win_v3.x.x.exe
文件大小: 约100-150MB

2. 运行安装程序

双击 ProtonVPN_win.exe 文件

3. Windows安全提示

如果出现"Windows已保护你的电脑"警告:
1. 点击"更多信息"
2. 点击"仍要运行"
3. 这是正常的安全提示，软件安全可靠

4. 用户账户控制(UAC)提示

如果弹出"是否允许此应用对设备进行更改":
点击"是"

5. 安装向导

1. 欢迎界面: 点击"Next"（下一步）
2. 许可协议: 选择"I agree"（我同意），点击"Next"
3. 安装位置: 使用默认位置，点击"Next"
4. 开始安装: 点击"Install"（安装）
5. 等待安装完成（约2-3分钟）
6. 完成: 点击"Finish"（完成）

第3步: 创建免费账户

首次启动Proton VPN:

1. 桌面双击 Proton VPN 图标
2. 欢迎界面会打开
3. 选择"Create Free Account"（创建免费账户）

填写注册信息:

Email: 输入您的邮箱地址
       - 可以使用QQ邮箱 (例: yourname@qq.com)
       - 或163邮箱 (例: yourname@163.com)
       - 或公司邮箱

Password: 创建密码
          - 至少8个字符
          - 包含字母和数字
          - 记住此密码！

Confirm Password: 再次输入相同密码

点击"Create Account"（创建账户）

验证邮箱:

1. 打开您的邮箱
2. 查找Proton VPN验证邮件
3. 点击邮件中的验证链接
4. 返回Proton VPN应用
5. 使用邮箱和密码登录

第4步: 连接VPN

登录后的界面:

1. 主界面显示地图和服务器列表
2. 免费版本可以使用3个国家的服务器

推荐服务器（按优先级）:

  • 🇯🇵 日本 (Japan) - 速度最快，稳定性好 ⭐⭐⭐⭐⭐
  • 🇸🇬 新加坡 (Singapore) - 速度快，延迟低 ⭐⭐⭐⭐⭐
  • 🇰🇷 韩国 (Korea) - 连接ELCANTO最优，稳定 ⭐⭐⭐⭐⭐
  • 🇳🇱 荷兰 (Netherlands) - 备用选择 ⭐⭐⭐

连接步骤:

1. 在服务器列表中找到"Japan"（日本）
2. 点击"Connect"（连接）按钮
3. 等待连接...（通常10-30秒）
4. 看到绿色"Connected"（已连接）✅
5. 任务栏会显示VPN图标（绿色盾牌）

第5步: 访问ELCANTO SCM系统

确认VPN已连接:

检查项:
□ Proton VPN显示"Connected"（已连接）
□ 服务器显示为"Japan"或其他选择的国家
□ 任务栏有绿色盾牌图标

打开浏览器:

推荐浏览器:
1. Google Chrome（最推荐）
2. Microsoft Edge
3. Firefox
4. 其他现代浏览器

访问网站:

1. 打开浏览器
2. 在地址栏输入: https://elcanto-scm.web.app
3. 按Enter键
4. 等待网页加载（首次可能需要10-20秒）

登录系统:

看到登录页面后:
1. 输入用户名: [公司提供的用户名]
2. 输入密码: [公司提供的密码]
3. 点击"登录"按钮
4. 进入系统主页 ✅

❗ 常见问题解决

问题1: VPN无法安装

错误: "Windows Defender已阻止"

解决方案:
1. 右键点击 ProtonVPN.exe
2. 选择"属性"
3. 勾选"解除锁定"
4. 点击"确定"
5. 再次双击安装

错误: "需要管理员权限"

解决方案:
1. 右键点击 ProtonVPN.exe
2. 选择"以管理员身份运行"
3. 点击UAC提示的"是"

问题2: VPN无法连接

错误: "Connection failed"（连接失败）

解决方案:
1. 尝试其他服务器
   - 日本 → 新加坡 → 韩国 → 荷兰
2. 检查网络连接是否正常
3. 关闭防火墙（临时测试）
4. 重启VPN应用
5. 重启电脑

问题3: 网站无法访问

错误: "无法访问此网站"

检查项:
□ VPN是否已连接（绿色图标）
□ 网址是否正确: https://elcanto-scm.web.app
□ 浏览器是否为最新版本

解决方案:
1. 确认VPN已连接
2. 清除浏览器缓存:
   - Chrome: Ctrl+Shift+Delete
   - 选择"所有时间"
   - 勾选"缓存"
   - 点击"清除数据"
3. 关闭浏览器，重新打开
4. 再次访问网站
