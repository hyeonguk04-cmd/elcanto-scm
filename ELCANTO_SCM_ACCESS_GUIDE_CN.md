## 🌟 方法1: 安装VPN (推荐) ⭐⭐⭐⭐⭐

### 安装步骤

#### 第1步: 获取VPN安装文件
从公司邮件、WeChat或QQ接收以下文件:
- `ProtonVPN.apk` (安卓手机)
- 或 `ProtonVPN.exe` (Windows电脑)

#### 第2步: 安装VPN (安卓手机)

1. **允许安装第三方应用**
   ```
   设置 → 安全与隐私 → 更多安全设置 → 安装未知应用
   找到"文件管理器"或"下载"，开启"允许来自此来源"
   ```

2. **安装APK文件**
   ```
   - 找到下载的 ProtonVPN.apk
   - 点击文件
   - 选择"安装"
   - 等待安装完成
   ```

3. **首次使用设置**
   ```
   - 打开 Proton VPN 应用
   - 选择 "Create Free Account" (创建免费账户)
   - 输入邮箱和密码
   - 验证邮箱
   - 登录
   ```

#### 第3步: 连接VPN

1. 打开 Proton VPN
2. 选择服务器国家:
   - 🇯🇵 **日本** (推荐 - 速度快)
   - 🇸🇬 **新加坡** (推荐 - 速度快)
   - 🇰🇷 **韩国** (推荐 - 连接ELCANTO最优)
   - 🇭🇰 **香港** (备用)
3. 点击 "连接" 按钮
4. 等待显示 "已连接" ✅

#### 第4步: 访问ELCANTO SCM

```
1. 确认VPN已连接 (显示绿色✅)
2. 打开浏览器 (Chrome、Safari、Firefox等)
3. 访问: https://elcanto-scm.web.app
4. 输入您的登录信息:
   - 用户名: [公司提供]
   - 密码: [公司提供]
5. 点击"登录"
```

### 常见问题解决

**问题1: VPN连接失败**
```
解决方案:
- 尝试其他服务器 (日本 → 新加坡 → 韩国)
- 检查网络连接
- 重启VPN应用
- 重启手机
```

**问题2: 网站加载很慢**
```
解决方案:
- 切换到日本或新加坡服务器
- 关闭其他正在使用的应用
- 等待1-2分钟
```

**问题3: 无法创建VPN账户**
```
解决方案:
- 使用公司邮箱地址
- 或使用QQ邮箱、163邮箱等
- 确保邮箱地址正确
```

## 🖥️ 方法2: TeamViewer 远程桌面 ⭐⭐⭐⭐⭐

**最简单的方法！无需VPN！**

### 什么是TeamViewer?
通过互联网连接到韩国办公室的电脑，在远程电脑上使用ELCANTO系统。

### 安装步骤

#### 第1步: 下载TeamViewer
```
手机下载:
- 安卓: 应用商店搜索 "TeamViewer"
- 或访问: https://www.teamviewer.com/cn/

电脑下载:
- 访问: https://www.teamviewer.com/cn/
- 点击"下载TeamViewer"
- 选择Windows或Mac版本
```

#### 第2步: 安装TeamViewer
```
1. 下载完成后打开文件
2. 选择"安装"
3. 接受协议
4. 完成安装
```

#### 第3步: 连接到韩国办公室电脑

**公司会提供给您:**
```
TeamViewer ID: [例如: 123 456 789]
密码: [例如: abc123]
```

**连接步骤:**
```
1. 打开TeamViewer应用
2. 在"伙伴ID"输入框中输入公司提供的ID
3. 点击"远程控制"
4. 输入密码
5. 点击"连接"
```

#### 第4步: 使用系统
```
连接成功后:
1. 您会看到韩国办公室电脑的桌面
2. 可以像使用自己的电脑一样操作
3. 打开浏览器
4. 访问 https://elcanto-scm.web.app
5. 使用您的账号登录
```

### TeamViewer 优势
- ✅ **完全免费** (非商业使用)
- ✅ **不需要VPN**
- ✅ **在中国稳定运行**
- ✅ **安装简单**
- ✅ **支持手机和电脑**

**问题: 连接失败**
