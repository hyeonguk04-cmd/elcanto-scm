# VPN 设置指南 - ELCANTO SCM 访问

## 方法1: 通过APK文件安装VPN (推荐)

### 安装步骤：

1. **下载VPN APK文件**
   - 从公司邮件中下载 ProtonVPN.apk 文件
   - 或从USB复制到手机

2. **允许安装未知来源应用**
   - 设置 → 安全 → 未知来源
   - 打开 "允许安装未知应用"

3. **安装APK**
   - 找到下载的 ProtonVPN.apk 文件
   - 点击安装
   - 按照提示完成安装

4. **使用VPN**
   - 打开 Proton VPN
   - 创建免费账户 (如果是第一次使用)
   - 选择服务器: 日本、新加坡、或韩国
   - 点击 "连接"

5. **访问网站**
   - VPN连接成功后
   - 打开浏览器
   - 访问: https://elcanto-scm.web.app
   - 使用提供的用户名和密码登录

2. **连接到韩国办公室电脑**
